# Expense Claim Anomaly Detector

AI-based fraud detection system for expense claims.

## Features
- Anomaly detection
- Risk scoring
- Explanation panel
- Dashboard
- Review queue
- CSV export

## Technologies Used
- Python
- FastAPI
- Streamlit
- Pandas
- scikit-learn

## Run Backend
uvicorn app:app --reload

## Run Frontend
streamlit run ui.py
